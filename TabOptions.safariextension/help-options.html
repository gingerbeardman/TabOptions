Tab Options Help

Options

New Tab Position & New Background Tab Position

These settings determine the positioning of new tabs and new background tabs relative to existing tabs. By default, Safari always positions new tabs at the end of the tab bar—that is, after all existing tabs. You can change this so that a new tab is positioned next to the current tab (on either side) or at the beginning of the tab bar instead.

Note: In Safari 5.1 and above, these tab positioning settings apply regardless of the method used to open a new tab. In Safari 5.0.x, the settings only apply when you open a new tab using a Tab Options hotkey.

Focus On Close

This setting determines which tab gets the focus (that is, becomes active) when you close the active tab.

Note: In Safari 5.1 and above, this setting applies regardless of the method used to close a new tab. In Safari 5.0.x, the setting only applies when you close a tab using a Tab Options hotkey.

Close Last Tab

This setting determines what happens when you close the last tab of the active browser window. Instead of closing the window (Safari's default behavior), Tab Options can keep the window open by emptying the last tab or loading a specified URL in it. Note: If you close the last tab by any method other than using a Tab Options hotkey, Tab Options cannot prevent Safari from closing the window, but it will immediately open a new window for you (if this setting is enabled).
